## Features

### 🐾 AI 陪伴聊天

像朋友一样和你聊天，而不是传统助手。

- 主动发起对话
- 随机闲聊
- 情绪化互动
- 长时间陪伴

---

### 💭 主动状态感知

AI Pet 会根据时间和状态主动出现。

例如：

- 深夜提醒早点休息
- 工作太久提醒喝水
- 长时间无操作主动搭话
- 摸鱼时间随机吐槽

---

### 🖥 桌面悬浮宠物

一个真正住在 macOS 桌面上的 AI。

- 悬浮显示
- 可拖拽
- 透明窗口
- 动态表情
- 随机动作

## macOS 安装提示

首次安装可能会出现：

> “Apple 无法验证开发者”

这是因为应用暂未进行 Apple 官方签名。

解决方法：

1. 右键 App
2. 点击「打开」
3. 再次确认打开

即可正常运行。

如果 macOS 提示无法验证开发者，请到“系统设置 > 隐私与安全性”里允许打开

## 自定义生成陪伴形象

当然可以！

首先生成你想要的任何形象：
- 格式：gif&png 建议使用gif素材，背景透明，可以循环播放动画，更加灵动有趣
- 大小：240x240px
- 素材包：放置到/.../TuTCat.app/Contents/Resources/Assets文件中，按照已分类的文件进行全部素材替换

同一种状态文件夹里的素材可以任意增加。当小猫进入任何一种非散步状态时，TuTCat会从相应文件夹的可用素材中随机播放，所以你甚至可以为一种状态设计很多种样子。
